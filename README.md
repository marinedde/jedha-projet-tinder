# Projet Tinder — Analyse des profils utilisateurs

**Jedha Bootcamp · Certification CDSD · Bloc 1**

## Contexte

Analyse exploratoire des données (EDA) des profils utilisateurs Tinder
afin de comprendre les comportements et caractéristiques des utilisateurs
de la plateforme de rencontres.

## Objectifs

- Explorer et nettoyer le dataset
- Identifier les tendances et patterns dans les profils
- Visualiser les distributions et corrélations clés

## Stack Technique

- **Python** (Pandas, NumPy)
- **Matplotlib / Seaborn** — visualisations
- **Jupyter Notebook / VS Code**
